 Smart Die System
 Project Description
Un système IA qui transforme les outils industriels en capteurs intelligents.

 Features
- Capteurs simulés (force, température, usure)
- Détection d'anomalies IA
- Alertes en temps réel

 How to Run
1. Install Python
2. Run: `python smart_die_simulation.py`
3. Watch the alerts!

 Technology
- Python
- TensorFlow Lite (pour la version complète)
- Streamlit (dashboard)
